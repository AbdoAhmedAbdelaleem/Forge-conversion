Class Reader

Utility class for parsing & reading SVF content from Model Derivative service or from local file system.

The class can only be instantiated using one of the two async static methods: Reader.FromFileSystem, or Reader.FromDerivativeService. After that, you can parse the entire SVF into memory using {@link parse}, or parse individual SVF objects using methods like readFragments or enumerateGeometries.

example

const auth = { client_id: 'forge client id', client_secret: 'forge client secreet' }; const reader = await Reader.FromDerivativeService('model urn', 'viewable guid', auth); const scene = await reader.read(); // Read entire scene into an intermediate, in-memory representation console.log(scene);

example

const reader = await Reader.FromFileSystem('path/to/svf'); // Enumerate fragments (without building a list of all of them) for await (const fragment of reader.enumerateFragments()) { console.log(fragment); }
